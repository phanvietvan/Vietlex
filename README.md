# Vietlex
Vietlex là nền tảng giúp người dùng ghi nhớ và học thuộc mọi loại văn bản bằng cách gõ lại thay vì đọc hoặc ghi chép. Bạn có thể dán bất kỳ nội dung nào như tiếng Anh, tiếng Việt, tài liệu hay bài thuyết trình, và học thông qua phương pháp typing-based memorization để ghi nhớ chủ động và hiệu quả hơn.
